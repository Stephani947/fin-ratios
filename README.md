## 📊 Understanding Financial Ratios

This software helps you analyze financial data by calculating ratios like:

- **Valuation Ratios:** Price to earnings, price to book, etc.
- **Profitability Ratios:** Return on assets, net profit margin, and others.
- **Risk Ratios:** Debt to equity, current ratio.
- **Composite Scores:** Altman Z-score, Piotroski F-score.
- Specialized ratios for SaaS, REIT, and banking sectors.

Each ratio provides insight into a company’s financial health and market value. You do not need to perform calculations yourself because fin-ratios handles all math automatically.
